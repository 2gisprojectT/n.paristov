Функциональное исследовательское тестирование поля поиска на сайте mail.ru

1.Количество выполненных сеансов - 4(пустое поле->обычный запрос->спец.символы(см.документацию)->
  ->максимальное количество символов)
2.Количество найденных проблем - 2(происходит поиск строки из спец.символов,ограничение 
  количества символов в поле нет)
3.Охватываемая область - проверка всех возможных значений для поля поиска
4.Процент от времени сеанса,посвященный настройке тестов - 25
5.Процент от времени сеанса,посвященный тестированию - 40
6.Процент от времени сеанса,посвященный анализу найденных проблем - 35

Не стал проверять поле на ввод какого-либо скрипта,поскольку,если проходит поиск по спец.символам,
то скрипт точно пройдет(буквы+спец.символы).

Документация:
Валидные символы - точка,запятая,двоеточие,тире,восклицательный знак,знак вопроса,точка с запятой
Специальные символы - символы, не являющиеся валидными символами
